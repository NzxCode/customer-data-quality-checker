# Customer Data Quality Checker

A foundational Python project focused on customer data validation and rule-based data quality checking.

---

## 📖 Overview
This project is designed to simulate how Data Scientists and AI Engineers handle raw customer data before using it for analysis or machine learning.

The main goal is to build a strong foundation in:
- Data validation
- Rule-based logic
- Problem-solving using Python

---

## 🎯 Objectives
- Understand Python fundamentals (variables, strings, numbers, boolean)
- Apply validation rules to real-world-like data
- Build a simple data quality checking system
- Develop a Data Scientist mindset

---

## 🧠 Key Concepts Used
- Variables
- String manipulation
- Numeric validation
- Boolean logic
- Conditional statements (if/else)
- Rule-based decision making

---

## ⚙️ Current Features
- Customer data input (name, email, age, etc.)
- Data type validation using `type()`
- Structured data representation
- Initial setup for validation logic

---

## 🚧 Work in Progress
- Email validation logic
- Price validation logic
- Data quality status (Valid / Invalid)
- Rule-based decision system
